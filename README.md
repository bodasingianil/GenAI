## 🧪 Featured Project: Blood Report Analyzer

### 🔍 Description
The **Blood Report Analyzer** is a Generative AI-powered application that helps users understand medical blood reports by converting complex values into **simple, human-readable insights**.

### ✨ Key Features
- Upload blood test reports (PDF or text)
- AI-based interpretation of health metrics
- Easy-to-understand explanations
- User-friendly web interface

### 🛠 Tech Stack
- Python
- Streamlit
- Generative AI / LLMs
- LangChain (for prompt orchestration)
